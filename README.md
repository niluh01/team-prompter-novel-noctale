Project ini dikembangkan sebagai bagian dari mata kuliah Software Development oleh tim Prompter.
Noctale dirancang dengan fokus pada:
- User Experience membaca yang nyaman
- Sistem manajemen novel yang fleksibel
- Interaksi antar pengguna dan penulis
